**Fitur Utama:**
- ⭐ AI Generator (Mode Simpel & Advanced)
- 💼 Brand Voice Management
- 📝 Caption History & Rating
- 📦 Order Request (Human Copywriter)
- 📊 Analytics
- 👤 Profile & Settings
- 🔔 Notifications

**Akses:**
- 200+ jenis konten
- 50+ platform support
- Multiple variations (1-20 caption)
- Auto hashtag
- Bahasa daerah (5 bahasa)
- 6 tone options

**Pricing:**
- Generate pertama: 5 variasi GRATIS
- 1 caption: GRATIS
- 5-20 captions: Rp 5.000 - Rp 15.000

## 🎮 USER JOURNEY

### Client Journey:
1. **Register** → Email verification
2. **Onboarding** → Tutorial AI Generator
3. **First Generate** → 5 variasi GRATIS
4. **Regular Use** → Generate caption (Mode Simpel/Advanced)
5. **Advanced Use** → Brand Voice, Multiple variations, Order human

### Operator Journey:
1. **Register** → Approval by guru/admin
2. **Training** → Complete tutorials & certification
3. **First Order** → Accept simple order, complete with guidance
4. **Regular Work** → Accept orders, use AI assistant, deliver quality
5. **Growth** → Build portfolio, increase rating, level up

### Guru Journey:
1. **Setup** → Create class, add students
2. **Training** → Train students on platform & copywriting
3. **Supervision** → Monitor orders, ensure quality
4. **Development** → Provide feedback, track progress
5. **Reporting** → Generate reports for school & parents

### Admin Journey:
1. **Setup** → Configure platform, payment, settings
2. **Launch** → Onboard users, marketing
3. **Monitor** → Track metrics, ensure quality
4. **Optimize** → Improve features, fix issues
5. **Scale** → Grow users, revenue, impact
