# 🚀 Bulk Content Generator - Quick Start Guide

## Untuk User (UMKM)

### Step 1: Akses Bulk Content Generator
```
Login → Dashboard → Klik icon 📅 "Bulk Content" di sidebar kiri
atau
URL: http://pintar-menulis.test/bulk-content
```

### Step 2: Generate Konten Baru
```
1. Klik tombol "Generate Konten Baru"
2. Isi form:
   ✏️ Nama Calendar: "Konten Instagram Maret 2026"
   📅 Durasi: Pilih "7 Hari" atau "30 Hari"
   📆 Tanggal Mulai: Pilih tanggal mulai
   🏷️ Kategori: Pilih kategori bisnis (Fashion, Food, dll)
   📱 Platform: Pilih platform (Instagram, TikTok, dll)
   🎭 Tone: Pilih tone (Casual, Professional, dll)
   📝 Brief: Jelaskan produk/bisnis kamu (detail = hasil bagus!)
3. Klik "Generate Konten"
4. Tunggu 2-3 menit (AI sedang bekerja...)
5. Done! Konten siap digunakan
```

### Step 3: Lihat & Edit Konten
```
Calendar View menampilkan:
- 📅 Tanggal & Hari
- ⏰ Jam Posting (otomatis)
- 🎯 Tema Hari Ini
- 📝 Caption Lengkap
- ✏️ Button Edit
- 📋 Button Copy

Untuk Edit:
1. Klik "Edit" pada konten yang mau diubah
2. Edit caption atau ubah jam posting
3. Klik "Save Changes"
```

### Step 4: Export & Gunakan
```
1. Klik "Export CSV"
2. Download file CSV
3. Buka di Excel/Google Sheets
4. Copy caption sesuai jadwal
5. Posting di platform kamu!

Atau:
- Import ke scheduling tools (Buffer, Hootsuite)
- Share dengan team
- Print untuk planning board
```

## Tips Menggunakan Bulk Content Generator

### 1. Brief yang Bagus = Hasil yang Bagus
```
❌ Brief Jelek:
"Jual baju"

✅ Brief Bagus:
"Jual baju wanita trendy untuk usia 20-35 tahun, 
harga terjangkau Rp 50k-150k, fokus ke kualitas 
bahan dan model yang up-to-date. Target market: 
mahasiswi dan pekerja kantoran yang suka fashion."
```

### 2. Pilih Durasi yang Sesuai
```
7 Hari (1 Minggu):
- Untuk testing pertama kali
- Untuk campaign pendek
- Untuk seasonal content

30 Hari (1 Bulan):
- Untuk planning bulanan
- Untuk konsistensi posting
- Untuk hemat waktu maksimal
```

### 3. Sesuaikan dengan Platform
```
Instagram:
- Tone: Casual/Inspirational
- Caption: Medium-long (500-1000 char)
- Hashtag: 10-15 hashtags

TikTok:
- Tone: Funny/Casual
- Caption: Short (100-200 char)
- Hashtag: 3-5 trending hashtags

Facebook:
- Tone: Professional/Educational
- Caption: Long (1000-2000 char)
- Hashtag: 3-5 hashtags
```

### 4. Edit Sesuai Kebutuhan
```
Konten yang di-generate adalah TEMPLATE.
Kamu bisa:
- Tambah info promo spesifik
- Ubah harga/diskon
- Tambah link produk
- Sesuaikan dengan event terkini
```

### 5. Konsisten Posting
```
Gunakan jadwal yang sudah di-generate:
- Pagi (09:00): Motivasi, Tips
- Siang (12:00): Edukasi, Product
- Malam (18:00): Promo, Testimoni

Konsistensi = Engagement naik!
```

## FAQ

### Q: Berapa lama proses generate?
A: 7 hari: 1-2 menit, 30 hari: 2-3 menit

### Q: Bisa edit hasil generate?
A: Bisa! Setiap konten bisa diedit individual

### Q: Bisa generate lagi kalau gak suka?
A: Bisa! Buat calendar baru dengan brief yang berbeda

### Q: Hasil generate sama semua?
A: Tidak! Setiap hari punya tema berbeda

### Q: Bisa untuk semua platform?
A: Ya! Instagram, TikTok, Facebook, Twitter, Shopee, dll

### Q: Gratis atau bayar?
A: Tergantung paket:
- Free: 5 caption/hari (gak bisa bulk)
- Starter: Bulk 7 hari
- Pro: Bulk 30 hari unlimited

### Q: Hasil generate bisa dipake langsung?
A: Bisa! Tapi recommended untuk review & customize

### Q: Bisa share dengan team?
A: Bisa! Export CSV dan share file-nya

### Q: Hashtag otomatis?
A: Ya! Setiap caption sudah include hashtag relevan

### Q: Bisa bahasa daerah?
A: Bisa! Pilih tone "Bahasa Jawa", "Bahasa Sunda", dll

## Troubleshooting

### Problem: Generate gagal
```
Solution:
1. Check internet connection
2. Check API key masih aktif
3. Coba brief yang lebih pendek
4. Refresh page dan coba lagi
```

### Problem: Hasil kurang sesuai
```
Solution:
1. Perbaiki brief (lebih detail)
2. Pilih tone yang lebih sesuai
3. Edit manual hasil generate
4. Generate ulang dengan brief baru
```

### Problem: Export CSV error
```
Solution:
1. Check browser allow download
2. Coba browser lain
3. Contact support
```
