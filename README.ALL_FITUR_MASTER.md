## 🎮 CARA MENGGUNAKAN

### Mode Simpel (Pemula)
1. Pilih jenis usaha
2. Isi nama produk/jasa
3. Isi harga (optional)
4. Pilih target market
5. Pilih tujuan (Closing/Branding/Engagement/Viral)
6. Pilih platform
7. Generate!

**Output:** 5 variasi GRATIS (generate pertama), 1 caption (generate berikutnya)

### Mode Advanced (Professional)
1. Pilih kategori (15+ pilihan)
2. Pilih subcategory (200+ pilihan)
3. Pilih platform
4. Isi brief detail
5. Pilih tone (Casual/Formal/Persuasive/Funny/Emotional/Educational)
6. Isi keywords (optional)
7. Pilih jumlah variasi (1/5/10/15/20)
8. Toggle auto hashtag
9. Pilih bahasa daerah (optional)
10. Generate!

**Output:** 1-20 variasi sesuai pilihan
