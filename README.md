> **Aplikasi Pembuat Caption Jualan Otomatis untuk UMKM Indonesia yang Ingin Closing, Bukan Cuma Estetik**

Platform marketplace yang menghubungkan client dengan operator copywriting profesional, didukung teknologi AI Google Gemini 2.5 Flash.

## ✨ Fitur Utama

### 🤖 AI Caption Generator

#### Mode Simpel (Untuk Pemula)
6 pertanyaan simpel, langsung jadi caption:
1. **Jenis Usaha** - 13 pilihan industri (Fashion, Makanan, Skincare, dll)
2. **Nama Produk** - Apa yang mau dijual
3. **Harga** - Berapa harganya
4. **Target Market** - Mau dijual ke siapa (Ibu muda, Remaja, Profesional, dll)
5. **Tujuan** - Closing / Branding / Engagement / Viral
6. **Platform** - Instagram / Facebook / TikTok / Shopee

**Output:**
- 5 variasi caption (default)
- Auto hashtag Indonesia
- Bahasa UMKM otomatis ("Kak", "Bun", "Gaes")

#### Mode Lengkap (Advanced)
Kontrol penuh untuk hasil maksimal:
- **Kategori:** Quick Templates / Industry Presets / 8 kategori lengkap
- **Jenis Konten:** 57+ subcategories
- **Platform:** Instagram, Facebook, TikTok, LinkedIn, Twitter
- **Brief/Deskripsi:** Detail produk dan keunggulan
- **Tone:** Casual, Formal, Persuasive, Funny, Emotional, Educational
- **Keywords:** Kata kunci spesifik (optional)
- **Bahasa Daerah:** Jawa, Sunda, Betawi, Minang, Batak (optional)
- **Generate Variations:** 5 variasi (gratis) atau 20 variasi (premium)
- **Auto Hashtag:** Hashtag trending Indonesia

**Output:**
- 5-20 variasi caption
- Hook generator (pembuka kuat 3 detik)
- CTA generator (10 variasi call to action)
- Headline generator (8 variasi)
- Quotes generator (motivasi & bisnis)
- Auto hashtag Indonesia

## 📦 Installation

### Requirements
- PHP 8.2 or higher
- Composer
- Node.js & NPM (optional)
- SQLite or MySQL

### Setup

1. **Clone Repository**
```bash
git clone <repository-url>
cd pintar-menulis
```

2. **Install Dependencies**
```bash
composer install
```

3. **Environment Setup**
```bash
cp .env.example .env
php artisan key:generate
```

4. **Configure .env**
```env
APP_NAME="Smart Copy SMK"
APP_URL=http://localhost

DB_CONNECTION=sqlite
# or MySQL:
# DB_CONNECTION=mysql
# DB_HOST=127.0.0.1
# DB_PORT=3306
# DB_DATABASE=pintar_menulis
# DB_USERNAME=root
# DB_PASSWORD=

GEMINI_API_KEY=your_gemini_api_key_here
```

5. **Database Migration**
```bash
php artisan migrate --seed
```

6. **Run Development Server**
```bash
php artisan serve
```

Visit: `http://localhost:8000`
