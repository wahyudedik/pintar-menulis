# Integrasi Kurikulum Smart Copy SMK

## 1. Pemetaan Kompetensi Dasar

### 1.1 Jurusan TKJ (Teknik Komputer dan Jaringan)

#### Mata Pelajaran: Administrasi Sistem Jaringan
**Kompetensi yang Diintegrasikan**:
- Instalasi dan konfigurasi web server (Apache/Nginx)
- Database management (MySQL/PostgreSQL)
- Monitoring sistem dan troubleshooting
- Backup dan recovery data

**Implementasi di Smart Copy**:
- Setup server untuk hosting platform
- Maintenance database client dan konten
- Monitoring uptime dan performance
- Backup rutin data client

#### Mata Pelajaran: Pemrograman Web dan Perangkat Bergerak
**Kompetensi yang Diintegrasikan**:
- HTML, CSS, JavaScript
- Framework PHP (Laravel)
- RESTful API
- Responsive design

**Implementasi di Smart Copy**:
- Develop dan maintain platform web
- Integrasi API AI (OpenAI, Claude)
- Membuat dashboard client dan admin
- Optimasi mobile experience

#### Mata Pelajaran: Produk Kreatif dan Kewirausahaan
**Kompetensi yang Diintegrasikan**:
- Analisis peluang usaha
- Perencanaan produk/jasa
- Strategi pemasaran
- Pengelolaan keuangan

**Implementasi di Smart Copy**:
- Riset kebutuhan UMKM
- Pricing strategy
- Marketing campaign
- Laporan keuangan bulanan

### 1.2 Mata Pelajaran Bahasa Indonesia

#### Kompetensi yang Diintegrasikan**:
- Teks persuasif dan argumentatif
- Kaidah kebahasaan (EYD, PUEBI)

- Struktur kalimat efektif

**Implementasi di Smart Copy**:

## 2. Jadwal Integrasi Pembelajaran

### 2.1 Struktur Mingguan

#### Senin - Rabu (Teori & Praktik di Kelas)
**13.00 - 14.30**: Pembelajaran normal sesuai kurikulum
**14.30 - 16.00**: Praktik Smart Copy (bergantian per kelas)

**Pembagian Tim**:
- **Tim A (TKJ)**: Maintenance teknis, development fitur
- **Tim B (Bahasa)**: Quality control, editing konten
- **Tim C (Marketing)**: Customer service, social media

#### Kamis - Jumat (Project-Based Learning)
**13.00 - 16.00**: Full operasional Smart Copy
- Morning briefing (15 menit)
- Eksekusi order (2 jam)
- Review & evaluasi (45 menit)

#### Sabtu (Shift Bergantian)
**09.00 - 12.00**: Shift 1 (Kelas X)
**12.00 - 15.00**: Shift 2 (Kelas XI)

### 2.2 Pembagian Tugas per Semester

#### Semester 1 (Kelas X)
**Fokus**: Pembelajaran dasar dan observasi
- Minggu 1-4: Training intensif (copywriting, platform, customer service)
- Minggu 5-8: Shadowing senior (observasi cara kerja)
- Minggu 9-16: Mulai handle order sederhana dengan supervisi

#### Semester 2 (Kelas X)
**Fokus**: Praktik mandiri dengan bimbingan
- Handle order basic package
- Belajar quality control
- Mulai terlibat dalam meeting client

#### Semester 3-4 (Kelas XI)
**Fokus**: Spesialisasi dan leadership
- Pilih spesialisasi (teknis/konten/marketing)
- Mentor untuk junior
- Handle order professional package
- Terlibat dalam strategi bisnis

#### Semester 5-6 (Kelas XII)
**Fokus**: Manajemen dan inovasi
- Posisi leadership (team lead, supervisor)
- Develop fitur baru
- Training untuk regenerasi
- Dokumentasi knowledge transfer

## 3. Metode Pembelajaran

### 3.1 Project-Based Learning (PBL)
**Prinsip**:
- Learning by doing
- Real client, real problem
- Collaborative work
- Reflective practice

**Implementasi**:
- Setiap order adalah mini-project
- Siswa bekerja dalam tim
- Presentasi hasil kerja mingguan
- Evaluasi dari client sebagai feedback

### 3.2 Teaching Factory
**Karakteristik**:
- Sekolah = pabrik/perusahaan
- Siswa = karyawan/entrepreneur
- Guru = supervisor/mentor
- Output = produk/jasa nyata

**Benefit**:
- Pengalaman kerja real
- Soft skill development
- Income generation
- Portfolio building

### 3.3 Blended Learning
**Kombinasi**:
- **Online**: Tutorial video, e-learning module, webinar
- **Offline**: Workshop, mentoring, team meeting
- **Hybrid**: Live streaming saat operasional, remote work

## 4. Sistem Penilaian Terintegrasi

### 4.1 Komponen Penilaian

#### Nilai Akademik (60%)
- **Pengetahuan** (20%): Quiz, ujian teori
- **Keterampilan** (40%): Praktik di lab, project

#### Nilai Smart Copy (40%)
- **Produktivitas** (15%): Jumlah order selesai
- **Kualitas** (15%): Rating dari client, revisi rate
- **Kolaborasi** (5%): Teamwork, komunikasi
- **Inovasi** (5%): Ide improvement, problem solving

### 4.2 Rubrik Penilaian Smart Copy

#### Excellent (90-100)
- Menyelesaikan >20 order/bulan
- Rating client 4.8-5.0
- Revisi rate <10%
- Aktif memberikan ide improvement
- Mentor yang baik untuk junior

#### Good (80-89)
- Menyelesaikan 15-20 order/bulan
- Rating client 4.5-4.7
- Revisi rate 10-20%
- Berkontribusi dalam diskusi tim
- Membantu junior saat diminta

#### Satisfactory (70-79)
- Menyelesaikan 10-14 order/bulan
- Rating client 4.0-4.4
- Revisi rate 20-30%
- Mengikuti SOP dengan baik
- Kooperatif dalam tim

#### Needs Improvement (<70)
- Menyelesaikan <10 order/bulan
- Rating client <4.0
- Revisi rate >30%
- Perlu bimbingan intensif

## 5. Modul Pembelajaran

### Modul 1: Fundamental Copywriting (4 minggu)
**Materi**:
- Apa itu copywriting dan kenapa penting
- Psikologi konsumen
- Formula copywriting (AIDA, PAS, FAB)
- Headline yang menarik
- Call-to-action efektif

**Praktik**:
- Analisis iklan sukses
- Latihan menulis caption
- A/B testing sederhana

### Modul 2: Platform & Tools (3 minggu)
**Materi**:
- Navigasi platform Smart Copy
- Cara menggunakan AI generator
- Quality control checklist
- Customer service basics

**Praktik**:
- Simulasi order
- Role play customer service
- Latihan quality control

### Modul 3: Social Media Marketing (3 minggu)
**Materi**:
- Karakteristik setiap platform
- Best practices per platform
- Hashtag strategy
- Content calendar planning

**Praktik**:
- Buat konten untuk berbagai platform
- Analisis kompetitor
- Planning content calendar

### Modul 4: Client Management (2 minggu)
**Materi**:
- Komunikasi efektif dengan client
- Handling complaint
- Upselling technique
- Building long-term relationship

**Praktik**:
- Role play berbagai skenario
- Case study real situation

### Modul 5: Business Operations (2 minggu)
**Materi**:
- Workflow management
- Time management
- Quality assurance
- Reporting & analytics

**Praktik**:
- Optimasi workflow
- Buat report bulanan
- Presentasi ke stakeholder
